Farenhyt IFP + HFSS

SERIES BLACK · IFP-2100 · KEYPAD & SOFTWARE
DAYS 1–2
IFP KEYPAD PROGRAMMING
DAY 1 — SLIDES · TRANSCRIPT · SELF NOTES
⚙
Defaulting — Pre-V7
  • Set panel ID via DIP switches → Power up
  • Enter installer code 123456
  • Do NOT press Enter after code on pre-V7
  • Navigate: 7 → Panel Programming → 7 → Restore Defaults
  • Scroll to YES → Enter twice (confirm + save)
  • Firmware stays the same — only program is wiped
🔐
Defaulting — V7+
  • Press Enter at main screen → Login Menu
  • Select Admin 1 → default code 0000000 (7 zeros)
  • Must press Enter after code on V7+
  • Navigate: 4 → Restore Defaults → YES → Enter twice
  • Firmware stays the same — only program is wiped
🔑
Password Requirements (V7+)
  • Admin 1 code must be exactly 7 digits
  • Cannot be same digit 7 times (e.g. 0000000)
  • Cannot be sequential (1234567 or 7654321)
  • Recommended: model # with leading zeros 0002100 or 0000300
  • V7.00.12+: Admin 1 change can update all 4 codes at once
  • Always write codes down — no backdoor on Series Black
⚠
Lost Password Recovery
  • 10 incorrect attempts → 30-second lockout
  • 11th failure → encrypted 20-char hex string displayed
  • Contact Technical Support with the hex string
  • Requires Password Agreement — signed & notarized by panel owner or manager (not tech/dealer)
  • Process can take 1–3 days
⚡
JumpStart
  • Scans all SLC devices & assigns types automatically
  • Incremental: builds on existing programming — does NOT erase
  • Removes devices no longer on SLC; keeps existing config
  • Monitor Modules default to Manual Pull — change manually
  • All inputs assigned to Zone 1 by default
  • Must select SLC protocol on launch — 1 protocol per panel
  • Only accessible through annunciator (not HFSS)
  • Already-assigned types won't be overwritten
📦
JumpStart Default Groups
  • Group 1 — all outputs (NACs, SLC relays, LEDs)
  • Group 998 — Onboard Relay 1 (Supervisory)
  • Group 999 — Onboard Relay 2 (Alarm)
  • Groups 997–999: do NOT add other devices here
  • Max 999 output groups on IFP-2100
🔢
Point ID Format
  • Format: MM : S/M/P NNN
  • Fixed IDs: 94=NVCM · 95=SKNICK · 97=Internal SLC · 98=Internal PSU · 99=Onboard Annunciator
  • S=Sensor | M=Module (IDP/SK) | P=Point (SD only)
  • NNN = 3-digit zero-filled address
  • I/O circuits: no letter — just the number 98:001
🗂
Inputs: Points vs Zones
  • Input Point = single addressable device
  • Input Zone = logical group of input points
  • Every point MUST be in exactly 1 zone
  • Multiple points can share the same zone
  • Max 999 zones; "Unused" = available address
  • 13 zone event types (alarms, supervisory, CO, water flow, Sonox 1 & 2…)
📢
Outputs: Points vs Groups
  • Output Point = single circuit or relay
  • Output Group = collection responding the same way
  • Each output point belongs to exactly 1 group
  • NEVER mix NACs and relays — different patterns required
  • Only 1 output pattern per group
  • Pattern 00=Constant | Pattern 19=System Sensor Sync
🗺
Mapping
  • Links input zones → output groups (action / reaction)
  • Any number of zones can map to 1 output group
  • 1 zone can map to any number of output groups
  • CANNOT be done through annunciator — HFSS only
  • Always pull program down after changes (Get Config)
🔧
Function Keys (Macros)
  • Hold F-key → installer code → press 1 to START recording
  • Record any dis/enable of points, zones, or groups
  • Hold same F-key → installer code → press 1 to END
  • Toggle: press once=activate, press again=deactivate
  • Latch through resets — great for annual testing
  • After recording: Get Configuration in HFSS to save macro digitally
  • Default: only Installer profile can activate F-keys
🔍
Built-in Troubleshooting Tools
  • SLC Device Locator: flashes LED on target — times out 10 min
  • Multiple Device Locator: up to 8 devices; both flash on double address
  • System shuts down during locate — press Enter to accept
  • I/O Point Control: toggle outputs / trip inputs; no timer; must be enabled in profile
  • Point Status: NORMAL · ALARM · CAL MAINT · CAL TRBLE
  • Detectors sampled every 3 hours; CAL TRBLE = out of NFPA 72
HFSS SOFTWARE
DAY 2 (PARTIAL) — SLIDES · TRANSCRIPT · SELF NOTES
💻
System Requirements
  • 64-bit Windows 10 or 11
  • Must have at least one active USB port
  • New panels: USB only — no DB9/serial port
  • USB 2.0 most stable; USB 3.0 may work on latest firmware
  • Only one program open at a time — best practice
  • Company-blocked USB? Contact IT to enable
📋
HFSS Menus Overview
  • Help → About: build date, Tech Support #, software version
  • Help → Licensing: log in to CLSS to enable programming license
  • Window: cascade / tile multiple programs (still use one at a time)
  • Options → Communication: select USB (only option)
  • Options → File: set default save location
  • Options → Panel Password: change passwords via software (HFSS 8.0.21.10 + FW 7.00.16+)
  • View: show/hide Queue, Toolbar, Status Bar
🔑
Changing Passwords via HFSS
  • Requires HFSS V8.0.21.10 or higher
  • Requires panel firmware V7.00.16 or higher
  • Options → Panel Password → enter Admin 1 code (default 0000000)
  • Click Get Panel Passwords → current passwords appear
  • Change all 4, click Set Panel Passwords
  • Invalid passwords shown in red with popup explaining requirements
  • Same 7-digit / no-repeat / non-sequential rules apply
📡
Status Bar & Connection
  • Status Bar: very bottom-right — real-time comm status
  • "Waiting on USB port" = panel detected, ready to communicate ✓
  • "No panel detected on USB port" = troubleshoot
  • Queue (bottom-center): shows all comm events
  • Successful Get Config: Status = Complete + progress 100%
  • "Panel not ready" = connection conditions not met
✅
Panel Connection Requirements
  • Panel must be at Normal or Trouble — main screen
  • ❌ Cannot connect during Fire Alarm
  • ❌ Cannot connect during Supervisory (active tampers = supervisory)
  • ❌ Cannot connect while in Keypad Programming menu
  • ❌ Cannot connect during SLC Initialization
  • SLC Short = only trouble that prevents connection on standalone
  • Open circuits, missing points, double address = OK to connect
🛠
USB Troubleshooting Steps
  • Unplug and re-plug cable · Try different USB port · Try different cable
  • Confirm Options → Communication is set to USB
  • Try USB 2.0 port instead of 3.0
  • Restart HFSS; try connecting before or after opening software
  • Power cycle the panel and/or PC
  • Make sure panel is powered on and at the main screen
  • Running Notifier? Close NFN Gateway in Task Manager
  • Expired CLSS license: still detects panel — only blocks Set Config
📁
File Menu — Get & Set Config
  • Get Configuration = Upload (panel → computer)
  • Set Configuration = Download (computer → panel)
  • New Configuration: blank file — panel only, no devices
  • Get Event History / Get Detector Status: pull logs to PC
  • Get Time / Set Time: sync panel clock to laptop
  • Report Header: company name, address, logo — export to Word/Excel/PDF
  • Always name your files! Auto-name uses date/time and gets confusing
  • Default Computer Access #: 123456 | Computer Code: 0
🌳
Directory Tree & Navigation
  • Top-left panel shows programming tree; +/– to expand/collapse
  • Right-click Site → Add Site, Add Panel, Delete Site, Rename Site
  • Max 32 sites total in one HFSS file
  • Right-click Panel → Add Panel, Add SBUS Module, Delete, Rename, Change Panel ID
  • Different Series Black models can be networked (2100, 300, 75)
  • Right-clicking is the primary way to interact — used constantly
⚙
Panel Parameters
  • Custom Banner: up to 40 chars, auto-centered, Normal mode only
  • SLC Protocol: IDP, SK/SS, or SD — 1 per panel
  • Water Flow Delay: 1–90 sec (global) — skip if device has built-in delay
  • Alarm Verification Time: 1–250 sec
  • Auto-Unsilence Time: 4 or 24 hrs — re-alerts if silenced but not fixed
  • Detector Sensitivity: set day/night with auto-switch start times
  • Sonox Names: custom label Zone Aux 1 & 2 (up to 20 chars)
📞
Communications Menu
  • Line Prefix: dial-out prefix ONLY — NOT the phone number (max 9 digits)
  • Phone Line Monitor: below 3V triggers phone line trouble
  • Line Disabled: for local-only systems or non-phone communicators
  • Dial Tone Detection Disabled: use with IP communicators (IP-DACT, etc.)
  • DHCP Enabled: auto IP from network; uncheck for static IP
  • AlarmNet supervision: every 5 min, 1 hr, 6 hr, or 24 hr
📥
Input Zones in HFSS
  • Right-click to add, label (up to 25 chars), or modify zones
  • Always label zones — unlabeled programs are a nightmare
  • Mapping check marks show which event types activate which outputs
  • Remove check marks for event types not in the zone
  • Elevator smoke example: isolate in own zone, map only recall relay to it
📤
Output Groups in HFSS
  • Right-click group # → Add New Group (set start # and quantity)
  • Group labels: up to 25 characters — always label
  • Right-click point type → change circuit type (NAC A/B, Aux Power, dry contact)
  • Right-click group column → assign point to different group
  • Select multiple rows → assign to same group or Auto-Increment
  • Export Points List → CSV for central station or inspector
🗃
Templates
  • Template = group of groups ("mega group")
  • Right-click in Template section → Add New Template
  • Move groups to Selected using arrows
  • Use to: map many groups at once OR disable with single F-key
  • Activation Cadence in output groups ONLY applies when using templates
🗺
Mapping in HFSS (3 Levels)
  • Zone Mapping: most common — detector/pull/water flow events per area
  • Panel Mapping: panel-level events (AC loss, battery) — mainly for LED annunciators
  • Site Mapping: ECS events, function key triggers, general alarm/supervisory
  • Right-click zone → Select Groups to Map → move to Selected → OK
  • Right-click zone → Remove Zone Mapping = NO UNDO
  • To add zone back: right-click existing zone → Show Zones to Map
  • Mapping is the most important part of Farenhyt programming
